\section{git 一些常见命令}
 \begin{itemize}
	\item git add 添加文件到暂存区
	\item git commit -m "first commit" 提交更改，从暂存区到当前分支。
	\item git push 正式提交到远程，谨慎操作。
	\item git status 查看当前状态。
	\item git reset --hard HEAD\^(或者commit的版本号)，回退，没有push之前。
	\item  git diff HEAD -- file 查看修改
	\item  git checkout -- file 回退
	\item git clone
\end{itemize}
